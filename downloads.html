Como instalar

Windows
  1. Baixe o instalador (.exe) oficial.
  2. Clique duas vezes e siga o assistente.
  3. Faça login e escolha as pastas para sincronizar.
macOS
  1. Baixe o .dmg e arraste o app para "Aplicativos".
  2. Abra o app (confirme a origem confiável, se solicitado).
  3. Faça login e selecione pastas favoritas para offline.
Linux

.deb (Ubuntu/Debian): sudo dpkg -i tcloud_x.y.z_amd64.deb && sudo apt -f install

.rpm (Fedora/RHEL): sudo rpm -i tcloud-x.y.z.x86_64.rpm

AppImage: chmod +x TCloud-x.y.z.AppImage && ./TCloud-x.y.z.AppImage

Ajuda rápida

O app não abre no Windows

Verifique se o Windows está atualizado e tente executar como administrador. Se persistir, reinstale.

macOS bloqueou a execução

Abra Preferências do Sistema → Segurança e Privacidade → "Abrir assim mesmo" para apps identificados.

Linux: dependências ausentes

Após dpkg -i rode sudo apt -f install para resolver dependências automaticamente.
