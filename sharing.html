Perguntas frequentes

Posso compartilhar sem criar conta?

Sim, via TCloud Transfer. Para controles extras (branding, auditoria), use uma conta TCloud.

O que acontece após a expiração?

O link deixa de funcionar. Você pode renovar ou criar um novo link a qualquer momento.

Como impedir downloads?

Use o visualizador seguro com bloqueio de download e/ou watermark. Ideal para revisões rápidas.

Zero‑Knowledge vale para links?

Com senha, o link pode aplicar criptografia no cliente (Zero‑Knowledge). Sem senha, os controles permanecem, mas ajuste a política de dados conforme sua oferta.
